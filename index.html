About.

  • 27 years old
  • From the Netherlands
  • Graduated in 'Art: Decoration and Restoration'
  • Worked with GML, C#, HTML, CSS, Javascript, Angular, p5.js library

Having spent most of my life pursuing art, my interest in programming came as a surprise. I find it offers both a breadth and depth, yet is more tangible and less subjective than art. It's a fresh wind I mean to explore further.

Currently, I am focused on improving my frond-end skills but I'm also very excited about Generative Art, Procedural Generation and Game Development. When I can find the time I paint on canvas and I'm curious to someday combine this with my interest in programming.

Projects.

Below you can find links to projects I've made during my programming journey. I started out in GML/GameMaker and after a lot of creating and experimenting I realized I really enjoy making things happen on the screen by typing code and chose to take it more seriously.

Shifting my focus toward C#, I started talking with people who program profesionally and worked on a few more projects.

Eventually I realized most of my enjoyment springs from working on visual projects or fidgeting with UI. Again I shifted my focus, this time toward Javascript and front-end, which suits me much better.

GameMaker.

My first projects were written in GML, using GameMaker Studio 2. I've made several small projects consisting of game mechanics that can be used in larger game-related projects.

Among other things, it gave me a basic understanding of if-statements, for-loops, switches and finite-state machines. It also taught me the importance of project scope and the dangers of feature creep.

Go to my itch.io.

MapGen Bronze (C#).

One of the bigger projects I have made is a 2D Map Generator, which was made as a C# Console Application using Simplex Noise to generate it's landmass and biomes. The generated maps can be saved (and loaded) as JSON files and can be navigated using the arrow-keys.

This project taught me my first lessons in C#, the importance of keeping my code organized and made me familiar with the use of libraries.

Go to Project Page.

Generative Art (p5.js).

My interest in generative art led me to try out the p5.js library. After a few experiments, I made a simple generator that uses the Random-Walker technique to generate and draw visuals (influenced by several adjustable variables) onto a HTML Canvas.

Working on this project taught more about organizing, refactoring and commenting my code.

Go to Live Project.

Dice Roller (Angular).

To learn more about Javascript frameworks I made a project in Angular 8, namely, a digital Dice Roller with a variety of die to choose from.

Making this Dice Roller showed me the value of Javascript frameworks, made me more familiar with HTML and CSS and was my first attempt at responsive design.

Go to Live Project.

MapGen Gold (p5.js).

When I began to become more comfortable with HTML, CSS and JS, I started wondering if I could recreate my Map Generator using the p5.js library. This went easier than expected and the project turned out far better than my C# Map Generator.

It uses Perlin Noise to generate it's maps, has a menu to manipulate the generation process and has different visual options. I'm not done with it yet and plan to add a lot more features.

The size of this project makes it a challenge to keep it organized but that's just a lesson to learn. Working on this, I gained more knowledge about HTML, CSS, Javascript and p5.js. I also learned more about binding UI to logic and dynamically creating HTML input elements.

Go to Live Project.

Template Canvas UI (HTML/JS).

Since I started making more HTML Canvas based projects, I decided to create a responsive template UI for it, which only needs to be hooked up to it's program's logic.

This project deepened my understanding of responsive design, and taught me more about modular design.

Go to Live Project.
